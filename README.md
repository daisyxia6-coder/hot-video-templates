# product-selling-makeup-skill
> 爆款视频内容拆解工具。爆款拆解模板的结构是按照爆款模板类型来的，不同的模板可使用的品类不一样。本版先做「效果对比」类型（竞品翻车/前后对比），后续持续更新其他类型。
## 快速开始
触发词：**拆解视频、分析视频、内容拆解、爆款拆解、效果对比**
用户提供视频ID或分享链接，自动完成以下全部流程：
1. 下载视频
2. 提取音频脚本
3. 截取关键帧
4. 识别爆款类型（竞品翻车 / 前后对比）
5. 按结构框架输出完整分镜脚本
6. **打包交付物**：`分镜脚本（Markdown）+ 关键帧截图（JPG 合集 zip）`
## 框架结构



效果对比型（一级结构）
├── 竞品翻车型（功效可外显）
└── 前后对比型（功效难外显）

## 适用模板
| 模板类型 | 核心特征 | 代表品类 |
|---------|---------|---------|
| 竞品翻车型 | 功效可量化/外显，优劣一眼能分辨 | 美妆工具、清洁剂、电动工具 |
| 前后对比型 | 需上脸/入口才看得出来，主观性强 | 口红、洗发水、服装 |
