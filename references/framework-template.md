# 美妆「上妆试色」内容框架模板

## 一、内容基础信息

| 字段 | 说明 |
|------|------|
| 产品名称 | （根据实际产品填写） |
| 产品品类 | 眼线笔 / 眉笔 / 睫毛膏 / 口红 / 粉底… |
| 目标用户 | 化妆新手 / 进阶级 / 某类肤质人群… |
| 视频时长 | 30s / 60s / 90s / 82s（示例） |
| 输出比例 | 9:16（抖音/小红书） |
| 核心叙事逻辑 | 双眼前后对比（画✅ vs 空白❌）全场实时变装 |

---

## 二、三段式内容框架

```
开场（0-15%）→ 试色演示（60-75%）→ 收尾CTA（15-20%）
```

### PART 1 ｜ 开头
**目标：** 制造反差感或期待感，留住用户

| 钩子类型 | 模板句式 |
|---------|---------|
| 素颜vs完妆反差 | 「不会画xx的老婆们看过来」 |
| 效果前置 | 「用它3步搞定！」 |
| 痛点直击 | 「xx星人必看！用它xx天搞定」 |
| 前后对比 | 「画vs不画，差的不是一点点」 |
| 工具导入 | 「xx想画好，工具一定要选对」 |

### PART 2 ｜ 试色演示（可自由组合）

**① 产品外观展示**
- 机位：产品特写 / 手持平铺
- 口播公式：类型+核心卖点
- 示例：「双头的」「弯月笔头」「软糯笔芯」「奶霜质地」

**② 质地测试**
- 机位：手臂/手背试色特写
- 口播公式：触感+使用感受
- 示例：「丝滑好上色」「不拉扯眼皮」「不干不卡」

**③ 上脸演示（核心环节）**
- 机位：侧面近景（展示手部动作）+ 眼部/唇部特写
- 口播公式：动作词 + 位置描述
- 示例：「睫毛根部撑起来，都给它填满色」

**④ 效果对比**
- 机位：同框左右对比 / 前后镜头切换
- 示例：大字「画✅ vs ❌没画」

**⑤ 持久度测试**
- 机位：手指蹭 / 纸巾擦 / 水喷
- 示例：「使劲蹭都蹭不掉」「夏天一整天不晕妆」

### PART 3 ｜ 收尾CTA

**四步连环：颜色推荐 → 套装内容 → 价格锚定 → 行动指令**

| 步骤 | 口播公式 | 字幕公式 |
|------|---------|---------|
| 推荐颜色 | 「推荐买xx色，xx场景都合适」 | 大字「xx色」「闭眼入不踩雷」 |
| 套装内容 | 「买一送x，到手x件套」 | 「自带xx+送xx共x件套 ✅」 |
| 价格锚定 | 「关键它才这个价格」 | 大字「到手x件套！」 |
| 行动指令 | 「想学的老婆们点击下方」 | 大字「点击下方 ↓」 |

## 三、字幕设计规范

| 位置 | 类型 | 示例 |
|------|------|------|
| 产品卖点 | 关键词+emoji | 「丝滑不拉扯 ✅」「弯月笔头 · 线条更细」 |
| 动作引导 | 短句+箭头 | 「填满根部 → 眼尾后拉」 |
| 效果强调 | 大字+对比 | 画 ✅ vs ❌ 没画 |
| CTA区 | 超大字 | 「02纯咖色」「闭眼入」 |

---

## 四、机位参考

| 环节 | 机位 | 画面重点 |
|------|------|---------|
| 开场 | 近景正脸/双眼同框 | 建立期待，露出反差 |
| 产品特写 | 极端特写/手持平铺 | 质感、颜色、细节 |
| 上妆演示 | 侧面近景+眼部特写 | 手部动作 + 效果同步 |
| 对比 | 双眼同框左右切 | 画✅ vs 空白❌ |
| CTA | 产品+手部/管身特写 | 价格 + 套装的视觉冲击 |

|------|-----------|-----------|---------|---------|
| ... | ... | ... | ... | ... |

**PART 2 ｜ 功能展示（Xs - Xs）**
...

**PART 3 ｜ 收尾CTA（Xs - Xs）**
...

### 第三部分：关键帧截图
- 工具：ffmpeg 每4秒截一帧（fps=1/4）
- 输出：JPG 格式
- 命名：frame_001.jpg, frame_002.jpg...

将所有截图打包为 zip 文件，文件名为「产品名_分镜脚本_关键帧.zip」，通过 message 工具发送给用户。
